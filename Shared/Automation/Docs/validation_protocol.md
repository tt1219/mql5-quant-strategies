# トレード戦略 承認プロトコル (Stage 1-3)

この文書は、新しい通貨ペアや銘柄を本番環境のポートフォリオに採用するために必要な、厳格な検証ステージを定義します。

## 🟢 Stage 1: 基本収益性と取引頻度
**目的**: ロジックに基本的な優位性があり、十分な取引チャンスがある銘柄を抽出します。

- **期間**: 直近の満1年間 (例: 2025.01.01 - 2025.12.31)
- **モデル**: 1分足 OHLC (始値・高値・安値・終値)
- **合格基準**:
    - **プロフィットファクター (PF)**: 1.5 以上
    - **総取引数**: 120回以上 (月平均10回以上)
    - **最大ドローダウン (DD)**: 10% 以下 (相対)
    - **リカバリーファクター**: 2.0 以上

## 🟡 Stage 2: 堅牢性と相場環境への適応
**目的**: 異なる相場局面でも破綻せず、スプレッドの変動に対しても過度に敏感でないことを確認します。

- **期間**: 特性の異なる3年間 (例: 2023, 2024, 2025)
- **モデル**: リアルティックに基づいた全ティック
- **合格基準**:
    - **プロフィットファクター (PF)**: 各年で 1.3 以上
    - **スプレッド耐性**: 通常スプレッド + 0.5 pips のペナルティを課しても利益がプラスであること
    - **安定性**: パラメーターをわずかに変更（例: ADX 30 → 32）しても、結果が大きく悪化しないこと

## 🔴 Stage 3: アイアンマン (長期生存試験)
**目的**: 10年間のヒストリカル検証により、ブラックスワン事象や大きなマクロ環境の変化を生き残れることを証明します。

- **期間**: 10年間 (例: 2016.01.01 - 2026.04.16)
- **モデル**: リアルティックに基づいた全ティック
- **合格基準**:
    - **生存**: 追証や口座破綻（Stop-Out）が一度も発生しないこと
    - **一貫性**: 10年間のうち、少なくとも8年間でプラス収益を上げていること
    - **ドローダウン耐性**: 最大ドローダウンが 20% を決して超えないこと
    - **主要イベント**: 2016年ブレグジット、2020年コロナショック、2022-2024年のインフレ急騰期を乗り越えていること

---
**運用ステータス**: **Stage 3** をクリアした銘柄のみが、`FullDeploy.mq5` による本番稼働の対象として承認されます。
