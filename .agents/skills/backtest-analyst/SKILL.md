# MT5バックテストレポート解析ガイド

MT5のストラテジーテスターが生成するレポート（HTML/XML）を読み解き、EAの真の実力を評価するための手順。

## 1. 信頼性の確認（最優先）

結果を見る前に、まず「データの質」を確認すること。

- **モデリング品質 (Modeling Quality)**: 99%（全ティック）または100%（リアルティック）であることを確認。これ未満の場合、スキャルピングEAの結果は無効。
- **グラフの連続性**: 資産曲線に不自然な垂直の線や空白がないか。あればヒストリーデータの欠損を示唆する。

## 2. 主要指標の評価基準

詳細は [metrics.md](references/metrics.md) を参照。

- **プロフィットファクター (PF)**:
  - 1.5以上: 優秀。
  - 1.2〜1.5: 合格点。
  - 1.0〜1.2: 改善が必要。
- **最大ドローダウン (Max Drawdown)**:
  - 10%以内: 非常に安全。
  - 20%以内: 標準的なリスク。
  - 30%以上: 運用には強い精神力が必要。
- **リカバリーファクター (RF)**:
  - 5.0以上: 早期の復帰が期待できる優秀な設定。

## 3. 取引分布の分析

- **勝率 vs リスクリワード (RR) 比**:
  - 勝率が低くても（30%）、RR比が高ければ（1:3以上）長期的に勝てる。
  - 勝率が高すぎる（90%以上）場合は、損切りをしない「ナンピン・マーチンゲール」の疑いを持つこと。
- **平均利益/平均損失**:
  - スキャルピングの場合、平均利益がスプレッドの3倍以上あるか。

## 4. 注意すべきレッドフラグ（危険信号）

- **取引数が少なすぎる**: 1年で10回未満などは統計的有意性がない。
- **特定の期間だけの爆益**: カーブフィッティング（過剰最適化）の可能性大。
- **全ティックでの全損**: OHLCモデルで勝てていても全ティックで負ける場合、そのEAはノイズに耐えられない。

## 使用方法

バックテスト結果が提示されたら、このスキルを起動して上記チェックリストに従い評価を生成せよ。
